## Usage

Use the mouse to look around and the following keys

| Key     | Action             |
|:-------:|:-------------------|
| W       | move forward       |
| A       | strafe left        |
| S       | move backward      |
| D       | strafe right       |
| Q       | turn left          |
| E       | turn right         |
| P       | pause (free mouse) |
| M       | toggle minimap     |
| esc     | quit               |

## Features
* A simple RayCasted FPS
* Using the SDL2 graphics library
* OpenCL acceleration
